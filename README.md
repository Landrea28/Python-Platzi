# 🐍 Python-Platzi 🚀

Repositorio creado para documentar mi ruta de aprendizaje, prácticas y proyectos relacionados con el lenguaje de programación **Python** 🐍 en Platzi.

---

## 📚 Resumen de Contenidos

A lo largo de este repositorio encontrarás distintos scripts y proyectos que reflejan mi avance, desde conceptos básicos hasta temas más avanzados:

1. 🛠️ **Fundamentos:** Datos flotantes, enteros, booleanos, listas, diccionarios y estructuras.
2. ⚙️ **Control de Flujo y Funciones:** Ciclos, parámetros, lambdas, excepciones y recursividad.
3. 📦 **Manejo de Archivos:** Prácticas de lectura y escritura de archivos JSON y CSV.
4. 🧩 **Programación Orientada a Objetos (POO):** Clases, métodos y cuentas bancarias (Ver carpeta `poo1/`).
5. 🧠 **Conceptos Avanzados:** Generadores, iteradores y listas por comprensión.
6. 🎯 **Proyectos Prácticos:** Minijuegos como Piedra, Papel o Tijera y ejercicios independientes (Ver carpeta `INDEPENDIENTE/`).

---

## 📂 Estructura del repositorio

El repositorio está organizado principalmente por archivos y carpetas temáticas correspondientes a los módulos de aprendizaje:

- `📁 poo1/` → Prácticas de Programación Orientada a Objetos.
- `📁 INDEPENDIENTE/` → Ejercicios y proyectos de lógica independiente.
- `📄 *.py` → Scripts en la raíz sobre conceptos específicos (ej. `Excepciones.py`, `Generador.py`, `Iteradores.py`).
- `📄 Datos` → Archivos `PythonPracticas.csv` y `PythonPracticas.json` para ejercicios de manejo de datos.
